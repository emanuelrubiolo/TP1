Herramientas que utilizo

Como desarrollador web con una sólida base en Ingeniería en Sistemas, tengo experiencia utilizando diversas herramientas y lenguajes de programación para crear soluciones eficientes y efectivas. Me siento cómodo trabajando con C#, lo que me permite desarrollar aplicaciones robustas tanto en entornos móviles como de escritorio. Mi conocimiento en Python y SQL me ha capacitado para manejar grandes volúmenes de datos y realizar análisis complejos, mientras que mi dominio de GitHub facilita la colaboración en equipo y el control de versiones de proyectos. Además, tengo una buena comprensión de tecnologías front-end como HTML, CSS, y JavaScript, lo que me permite diseñar interfaces de usuario atractivas y funcionales. También he trabajado con MongoDB, que me ayuda a gestionar bases de datos NoSQL de manera eficiente, adaptándome a las necesidades de proyectos más dinámicos. Estas habilidades me permiten abordar problemas de desarrollo web desde múltiples ángulos, siempre con el objetivo de entregar productos de alta calidad.
