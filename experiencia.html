Proyectos realizados

Proyecto: Sensor de llenado de cestos de basura con IoT (Actual)

Desarrollé un sistema de monitoreo de llenado de cestos de basura usando sensores ultrasónicos y un microcontrolador ESP32. El sensor mide el nivel de basura y envía datos a la nube en tiempo real. Cuando el cesto está casi lleno, el sistema envía alertas a dispositivos móviles, optimizando la recolección y reduciendo costos. Este proyecto facilita la gestión de residuos y mejora la limpieza de espacios públicos.

Proyecto: Plataforma de gestión de reservas para restaurantes (2023)

Desarrollé una plataforma web para la gestión de reservas en restaurantes, diseñada para mejorar la experiencia del usuario y optimizar la operación de los restaurantes. La aplicación permite a los clientes ver la disponibilidad de mesas en tiempo real, realizar reservas y recibir confirmaciones instantáneas por correo electrónico o SMS. En el lado del restaurante, la plataforma proporciona un panel de control intuitivo donde los gerentes pueden gestionar las reservas, asignar mesas, y ajustar la disponibilidad en función de la demanda. También incluye un sistema de análisis que muestra patrones de reservas, lo que permite a los restaurantes optimizar su capacidad y mejorar el servicio al cliente. Utilicé tecnologías como React para el front-end, Node.js y Express para el back-end, y MongoDB para la base de datos, asegurando una experiencia rápida y fluida tanto para los clientes como para los administradores del restaurante.
