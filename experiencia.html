Proyectos realizados

Proyecto: Sensor de llenado de cestos de basura con IoT (Actual)

Desarrollé un sistema de monitoreo de llenado de cestos de basura usando sensores ultrasónicos y un microcontrolador ESP32. El sensor mide el nivel de basura y envía datos a la nube en tiempo real. Cuando el cesto está casi lleno, el sistema envía alertas a dispositivos móviles, optimizando la recolección y reduciendo costos. Este proyecto facilita la gestión de residuos y mejora la limpieza de espacios públicos.

Proyecto: Plataforma de gestión de reservas para restaurantes (2023)

He creado una plataforma web para facilitar la gestión de reservas en restaurantes, que mejora tanto la experiencia del usuario como la eficiencia operativa del restaurante. Los clientes pueden ver en tiempo real la disponibilidad de mesas, hacer reservas, y recibir confirmaciones inmediatas por correo o SMS. Para los gerentes, la plataforma ofrece un panel de control fácil de usar para gestionar las reservas, asignar mesas, y ajustar la disponibilidad según la demanda. También incorpora un sistema de análisis que ayuda a optimizar la capacidad del restaurante y mejorar el servicio. Utilicé tecnologías como React, Node.js, Express, y MongoDB para asegurar una experiencia rápida y fluida.
